Roadmap¶

For the OpenDP roadmap, see https://opendp.org/roadmap

Kanban Board¶

The OpenDP Development board is used to track progress of the OpenDP library and related components/systems.

Issue Tracker¶

The main issue tracker for the OpenDP library can be found at https://github.com/opendp/opendp/issues

Other projects in the OpenDP Commons have their own issue trackers.

Getting Help¶

To discuss your contribution with OpenDP developers, please start a thread on GitHub Discussions.

Other OpenDP Projects¶

This guide is oriented toward contributing to the OpenDP library but you can find other OpenDP projects to contribute to in the OpenDP Commons section of the User Guide.

In addition, you can contribute by participating in the OpenDP community by joining the mailing list and attending events.

If you have an interest in improving this documentation, please see Documentation.
